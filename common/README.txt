Ottawa Linux Symposium 'common' directory
=========================================

This directory contains common files used by all OLS paper authors.
Please do not modify any of this files without consulting the OLS
formatting team in advance.

* Rules.mk
  Common build infrastructure for all OLS papers

* ols.cls
  The Ottawa Linux Symposium house style, thanks Zack Weinberg

* cprog.sty
  Directly include programs in C, C++, Pascal, and Modula-2

* notfloats.sty
  Tables and Figures that Really Appear HERE (not floating)

* underscore.sty
  Make the "_" character print as "\textunderscore" in text.

* zrl.sty
  Alternative to the url package
